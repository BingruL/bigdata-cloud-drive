智能云盘 用户使用文档

本文档面向首次使用智能云盘系统的用户，介绍从登录、上传、检索，到 AI 摘要、推荐与看板的完整使用流程。所有功能均可在浏览器中直接使用，无需额外客户端。

01快速开始

三步进入系统，开始你的第一次文件上传与检索。

STEP 1

打开应用

浏览器访问 http://localhost:5000/app，或点击引导页右上角「立即体验」。

STEP 2

登录账号

使用已有账号登录，或点击「注册」创建新账号。也可直接使用下方测试账号。

STEP 3

上传文件

进入「我的文件」，点击「上传」按钮或拖拽文件至页面即可开始使用。

若系统首次启动，请确认 HBase Thrift (9090) 与 HDFS NameNode (9870) 服务已运行，并通过 python run.py --seed 初始化测试数据。

02账号与登录

系统提供普通用户与管理员两种角色。管理员可查看全量日志、用户与系统配额。

注册新账号

登录页点击「注册」标签，填写用户名与密码即可创建账号。用户名在系统中唯一，请避免使用已存在的名称。

登录凭证以 JWT 形式存储在浏览器 localStorage 中，请勿在公共设备上勾选长期登录；退出登录会清除所有本地会话。

03文件管理

「我的文件」是使用频率最高的模块，支持目录浏览、上传、下载、预览、重命名、移动、删除等完整 CRUD 能力。

目录与整理

  • 点击「新建文件夹」可在当前目录创建子目录，面包屑可快速返回上级；
  • 文件和文件夹都支持重命名、移动到根目录或指定目标文件夹；
  • 同一目录下出现重名时，系统会自动生成可用名称，避免覆盖已有内容。

上传文件

  • 点击页面右上角「上传」按钮，可一次选择多个文件；
  • 或直接将文件拖入文件列表区域，松开即开始上传；
  • 上传过程会显示进度条，成功后自动刷新列表。

下载与预览

  • 下载：点击文件行的下载图标，浏览器会直接保存文件；
  • 预览：文本、图片、Markdown 和 PDF 支持在线预览，无需下载；
  • 详情：点击文件名可查看文件大小、MIME 类型、上传时间、下载次数等元信息。

公开链接

文件主可点击「公开链接」生成对外下载页，支持设置过期天数和可选提取码；链接可复制给未登录用户下载，也可在弹窗中随时撤销。

搜索与筛选

顶部搜索框支持按文件名模糊匹配，筛选下拉可按类型（文档、图片、视频等）过滤，结果实时更新。

私有 / 共享标识

每行文件名旁会显示一枚状态徽标：私有 表示仅你本人可见；共享 表示已分享到一个或多个群组，组内成员可下载与预览。文件默认创建为私有，需要手动通过「分享到群组」操作改变可见范围，详见 第 7 节 群组与共享。

04批量操作与排序

文件表支持全选、多选以及列头点击排序，方便处理大批量文件。

批量操作

  • 勾选文件前的复选框，或通过表头复选框一次性选择当前页所有文件；
  • 选中后顶部会出现批量工具条，可批量下载或删除；回收站页面下还提供批量恢复与批量彻底删除；
  • 批量删除的文件会先进入回收站，30 天内可恢复。

列排序

点击「文件名 / 大小 / 上传时间 / 下载次数」等列头可切换升降序。当前排序列会以高亮箭头标示。

05AI 摘要与标签

文档上传完成后可一键调用 LLM 生成内容摘要与自动标签，帮助快速理解与检索。

  • 在文件行展开操作菜单，点击 AI 摘要，系统会读取文件正文并调用 LLM 生成 3–5 句话的中文摘要；
  • 摘要生成后会写入文件元数据，再次查看无需重复调用；
  • 自动标签 会基于文档内容产出 3–8 个主题关键词，可用于后续筛选与关系图谱。
AI 功能依赖 AI_API_KEY 环境变量。未配置 Key 时该菜单会显示「未启用」。支持任意 OpenAI 兼容的 API 端点。

06文件关系图谱

把散落的资料重新连回一张图。基于文件标签与共同访问行为构建力导向网络。

  • 在「我的文件」页切换到 关系图谱 视图；
  • 节点代表文件，连线代表共享标签或关联访问，线越粗表示关系越强；
  • 支持拖拽、缩放，点击任意节点可高亮其直接邻居并在侧栏显示文件详情。

07群组与共享

群组是本系统中分享文件的基本单位。所有共享行为都在群组语境下发生：你只能把自己的文件分享给你已加入的群组，组内任何成员都能在「群组共享」页面看到这些文件。

我的群组

  • 侧栏点击「我的群组」打开群组管理页，列表展示你已加入或创建的所有群组；
  • 点击「创建群组」填写名称与描述即可建立新群组，创建者自动成为 owner；
  • 点击任意群组卡片进入详情页，可查看成员列表、加入时间与角色；
  • owner 与 admin 可在详情页通过用户名添加成员，或将非 owner 成员移出群组；
  • 普通成员可随时点击「退出群组」离开；owner 可点击「解散群组」删除整个群组（此操作会清理所有成员关系，已分享到该群组的文件将失去对组员的访问权）。

把文件分享到群组

  • 在「我的文件」中找到要分享的文件，展开操作菜单点击「分享」打开分享弹窗；
  • 弹窗会列出你所在的全部群组，勾选目标群组后保存即可——可一次勾选多个；
  • 取消所有勾选并保存等同于「取消分享」，文件状态会变回 私有；
  • 分享操作只更新元数据，不会复制文件内容，对存储空间无额外消耗。

群组共享

侧栏「群组共享」页展示 其他成员 分享到你所在任意群组的文件，可直接下载、预览，但无法删除或重命名（所有权仍属于原作者）。该页面与「我的文件」相互独立，自己分享出去的文件仍只在「我的文件」中显示。

权限规则：除文件所有者与管理员外，只有当文件被显式分享到 访问者所在的群组 时，下载 / 预览 / 元信息查询接口才会放行；其他情况一律返回 403。

08最近访问

系统会自动记录最近打开、下载或预览过的文件，并按访问时间倒序展示。

  • 侧栏点击「最近访问」进入该页面；
  • 每条记录显示文件名、操作类型（查看 / 下载 / 预览）与时间；
  • 可一键跳转回原文件所在目录。

09回收站与存储配额

删除不会立即清除 HDFS 数据，所有删除均先进入回收站并保留可恢复窗口。

回收站

  • 恢复：选择文件或文件夹点击「恢复」，内容回到原目录并恢复元数据；
  • 彻底删除：从 HDFS 与 HBase 同时移除，不可恢复；文件夹会递归清理子项；
  • 清空回收站：管理员可一键清空所有已删除内容。

存储配额

侧栏底部的容量条显示当前用户已用空间与配额上限。默认配额为普通用户 10 GB、管理员 200 GB，可通过后端环境变量覆盖。达到 80% 时容量条变为警示状态；超过配额时上传会被拒绝。

10数据看板

侧栏「数据看板」由服务层即时聚合 HBase 常规指标，并融合 Spark Structured Streaming 写入的实时面板数据，同屏展示当下系统活跃度与历史趋势。

实时面板（顶部）

  • 页面顶部的实时区按 2 秒 一次轮询 /api/stats/realtime，展示动作计数 / 热门文件（5 分钟窗口）、活跃用户（10 分钟窗口）以及最新 30 条事件流；
  • 右上角的状态指示灯：绿色「在线」表示 Spark Streaming 作业正在消费 Kafka，灰色「离线」表示当前未启用 Kafka / Streaming（看板其它部分仍可正常使用）；
  • 实时数据来自 Kafka topic cloud_drive_events，由 spark_jobs/streaming_stats.py 在 foreachBatch 中维护滚动窗口并写回 cloud_drive_stats。

常规统计

  • 文件类型分布：饼图展示各类型（文档 / 图片 / 视频 / 其他）占比；
  • 上传趋势：折线图显示最近 7 / 30 天的每日上传量；
  • 存储使用：条形图展示各用户空间占用；
  • 热门文件：按下载量降序列出 Top 10 文件。
常规图表由 Flask 统计服务直接扫描 HBase 后按当前用户视角聚合，登录即可使用；Spark 批作业会把离线分析结果写入 cloud_drive_stats，主要用于课程批计算演示与扩展。要启用实时面板需先 docker compose -f docker-compose.kafka.yml up -d 并启动 Streaming 作业。

11群组智能推荐

推荐结果在 群组语境 下计算：候选池只来自分享进你所在群组的文件，行为日志只取群组成员的操作。这意味着——你必须先加入或创建至少一个群组，推荐才会有内容。

  • 侧栏点击「推荐」进入推荐页，顶部会标注当前推荐的作用范围；
  • 系统提供三类推荐：热门文件（按下载与浏览频次排序）、个性化推荐（基于你近期行为与组内文件标签匹配）、相似用户也在看（基于 Jaccard 相似度找出兴趣相近的组员，推荐他们偏好的文件）；
  • 若你尚未加入任何群组，推荐页会提示 scope: no_group 并引导你前往「我的群组」创建或加入；管理员视角下推荐基于全站文件计算；
  • 推荐结果由 spark_jobs/recommendation.py 离线计算，可由管理员定期重跑刷新。

12管理员后台

仅 admin 角色可见，侧栏会额外出现「操作日志」入口。

  • 操作日志：查看全量上传 / 下载 / 删除 / 登录 / 分享 / 群组等操作记录，按时间倒序展示，可用于审计与排查；
  • 管理员同时拥有跨用户访问权：可下载、预览任意用户的文件，并对任意群组执行成员管理与解散；
  • 用户管理、配额调整、系统概览等功能尚未在前端实现，可通过后端脚本（如 scripts/data_lifecycle.py）或直接操作 HBase 完成。

13常见问题

上传失败，提示「HDFS 连接失败」？

请确认 HDFS NameNode 已在 9870 端口运行，并检查后端 HDFS_URL 环境变量配置是否正确。使用 curl http://localhost:9870/webhdfs/v1/?op=LISTSTATUS 可快速自检。

AI 摘要按钮点击后没有反应？

说明未配置 AI_API_KEY 或该文件不在支持的文本类型范围内。当前 AI 功能支持 txt / md / docx / pdf 等文本格式。

忘记了登录密码怎么办？

普通用户请联系管理员重置；管理员密码由启动时的 ADMIN_PASSWORD 环境变量或默认 admin123 控制。

数据看板没有数据？

常规看板数据由后端实时扫描 HBase 聚合。若顶部实时面板没有数据，请确认 Kafka 与 Spark Structured Streaming 已启动；未启用时不影响其它图表。

文件关系图谱显示为空？

关系图谱需要至少两份文件共享标签。请先通过 AI 自动标签或手动添加标签，图谱便会自动构建。

推荐页一直没有内容？

推荐结果的范围限定在你所在的群组。若你尚未加入任何群组，请先到「我的群组」创建或加入一个群组，并确保群组内已有成员分享过文件，推荐才能产出结果。

看板顶部一直显示「离线」？

实时面板依赖 Kafka 与 Spark Structured Streaming。需先启动 Kafka（docker compose -f docker-compose.kafka.yml up -d），将后端的 KAFKA_ENABLED 设为 1 并重启，再运行 spark-submit spark_jobs/streaming_stats.py。未启用时不影响其它功能。

把文件分享给某人，他在「群组共享」里看不到？

分享是 面向群组 的，不是面向个人。请确认对方是你分享目标群组的成员；若不是，请先在「我的群组」中将其加入群组。
